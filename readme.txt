Show your Acumbamail signup forms easily in your Wordpress pages through a widget.

== Description ==

This plugin make a widget to show your Acumbamail signup form easily in your blog. The signup form can be sorted as you want, and you can modify whatever you want of it.

Este plugin crea un widget para mostrar el formulario de inscripción a una lista de Acumbamail. Permite también modificar el texto de los campos así como ordenarlos.

== Installation ==

How to install?

1. Upload the plugin folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Configure your Acumbamail account inside Wordpress Admin

(Remember that you must have cURL installed on your server)

More info:
https://acumbamail.com/en/integrations/wordpress/

¿Cómo se instala?

1. Sube la carpeta del plugin al directorio `/wp-content/plugins/`

2. Activa el plugin mediante el menú de plugins incluido en wordpress

3. Configura tu cuenta y tu lista de acumbamail desde el panel de administración de Wordpress

(Recuerda que debes tener instalado cURL en tu servidor)

Más información:
https://acumbamail.com/integraciones/wordpress/
